# Simple Task List App

A minimal task list application for our group assignment.

## How to Run
Just open `index.html` in any web browser.

## Current Features
- Add new tasks
- View list of all tasks

## Your Task
Each person must add ONE new feature in their own branch.
Possible features:
- Due dates
- Delete tasks
- Priority levels
- Search/filter
- Mark as complete
- Categories/tags
- Save to local storage

## Branch Strategy
1. Create your branch: `git checkout -b feature/your-feature-name`
2. Implement your feature
3. Commit and push
4. Open merge request to main
